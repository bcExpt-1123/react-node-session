# RN-Rock-Paper-Scissors
Simple Rock Paper Scissors Game Made in React Native runs on Android, iOS, Web, MacOS, Windows

# How to develop

Clone the repo
```
git clone https://github.com/ShivamJoker/RN-Rock-Paper-Scissors
```

Change directory 
```
cd RN-Rock-Paper-Scissors
```

Install packages
```
npm install
```

Run on Android
```
npm run android
```

Run on iOS
```
npm run ios
```
